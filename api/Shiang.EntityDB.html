    Class EntityDB

    This class stores the following information of an entity 1) stats (TODO) 2) items to serialise the inventory 3) abilities to serialise the ability container

    At runtime, the data in the database are serialised and the actual classes will hold those information. Once that happen, the database will be cleared at runtime and only persist the data again when the game exit. (Or the game is explicitly saved by the user?)
